### 2️⃣ 资金管理（2.5倍马丁格尔）

#### ✅ 核心公式
```python
# 倒推首注金额
s0 = R × B × (Q - 1) / (Q^(N+1) - 1)

# 第 k 次投注
stake = s0 × Q^k

# 应用限制
stake = max(stake, 1.0)          # 最小 1 USDT
stake = min(stake, B × 0.05)     # 最大 5%
```

#### ✅ 参数配置
```python
R = 0.30        # 单序列占资金 30%
Q = 2.5         # 倍率 2.5
N = 4           # 最大连败 4 次
MAX_ONE = 0.05  # 单场最大 5%
```

#### ✅ 连败检测

### 3️⃣ 日内风控

#### ✅ 自动触发的保护机制
```python
1. 单日最大亏损 20%
   → 达到后停止当天所有投注

2. 单日盈利目标 5%
   → 达到后提醒收工（可配置停止）

3. 连败上限 4 次
   → 超过后强制结束序列

4. 余额保护
   → 单场不超过 5%
   → 始终保留 1 USDT
```

## ⚠️ 风险警告（必读）

### 马丁格尔策略的本质风险

1. **连败概率**
   ```
   假设 50% 胜率: 连败4次 = 6.25%
   假设 60% 胜率: 连败4次 = 2.56%
   ```

2. **资金需求**
   ```
   100 USDT 本金最坏需要 14.54 USDT (14.5%)
   但如果连败更多次，风险会指数增长
   ```

3. **盈利要求**
   ```
   需要至少 60% 真实胜率才能长期盈利
   赔率有 5-10% 抽水，实际要求更高
   ```

### 使用原则（必须遵守）

1. ❗ **只用闲钱** - 承受得起全部损失
2. ❗ **严格止损** - 达到限制立即停止
3. ❗ **及时提现** - 达到盈利目标锁定
4. ❗ **记录数据** - 每天统计胜率
5. ❗ **理性投注** - 不要情绪化追损

def filter_matches():
    # 1. 获取 API 数据
    competitions = fetch_events()

    # 2. 逐层筛选
    for comp in competitions:
        # 基础筛选
        if not in CORE_LEAGUES: continue
        if not in time_window: continue
        if not in odds_range: continue

        # 高级筛选
        if home_win_odds > MAX: continue
        if not trend_ok: continue

        # 计算评分

        if score < MIN: continue

        candidates.append(...)

    # 3. 排序 + 30分钟控制
    candidates.sort(key=lambda x: x['score'])

### bankroll.py - 资金计算

```python
def next_bet_amount(logs, balance):
    # 1. 日内风控
    day_pnl = calc_today_pnl(logs, balance)
    if day_pnl <= -DAILY_LOSS_LIMIT:
        return 0.0, 0, "止损"

    # 2. 连败检测
    loss_streak = current_loss_streak(logs)
    if loss_streak > N:
        return 0.0, loss_streak, "超连败"

    # 3. 计算金额
    s0 = calc_base_stake(balance)
    stake = s0 * (Q ** loss_streak)

    # 4. 应用限制
    stake = max(stake, MIN_BET)
    stake = min(stake, balance * MAX_ONE)

    return stake, loss_streak, None
```
