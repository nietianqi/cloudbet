# 联赛模式配置说明

## 📊 联赛统计

已配置 **184 个全球联赛**，分为 3 个等级：

| 等级 | 数量 | 说明 |
|------|------|------|
| **TIER_1** | 38个 | 核心顶级联赛（五大联赛 + 主流联赛） |
| **TIER_2** | 115个 | 全球前100国家一级、二级联赛 |
| **TIER_3** | 31个 | 国际赛事和杯赛 |
| **总计** | **184个** | 覆盖全球主要足球联赛 |

---

## ⚙️ 配置方法

编辑 `config.py` 文件：

```python
# 联赛选择模式
LEAGUE_MODE = 'TIER_1'  # 可选: 'TIER_1' | 'EXPANDED' | 'ALL'
```

### 三种模式说明

#### 1️⃣ TIER_1 模式（默认，推荐）

```python
LEAGUE_MODE = 'TIER_1'
```

**包含：** 38 个核心顶级联赛

**优点：**
- ✅ 最高质量，信息最充分
- ✅ 强队让1球最靠谱
- ✅ 胜率最稳定
- ✅ 推荐新手使用

**包括：**
- 五大联赛（5个）：英超、西甲、德甲、意甲、法甲
- 欧洲主流（17个）：荷甲、葡超、比甲、土超等
- 美洲主流（9个）：巴甲、阿甲、墨超、美职联等
- 亚洲主流（7个）：日职联、韩K联、中超、沙特等

#### 2️⃣ EXPANDED 模式（扩展）

```python
LEAGUE_MODE = 'EXPANDED'
```

**包含：** 153 个联赛（TIER_1 + TIER_2）

**优点：**
- ✅ 更多投注机会
- ✅ 覆盖全球主要联赛
- ⚠️ 需要更多数据验证

**新增：**
- 五大联赛次级（5个）：英冠、西乙、德乙等
- 欧洲次级联赛（15个）
- 非洲联赛（17个）
- 中东欧联赛（16个）
- 亚洲其他联赛（30个）
- 中北美联赛（8个）
- 等等...

#### 3️⃣ ALL 模式（全部）

```python
LEAGUE_MODE = 'ALL'
```

**包含：** 184 个联赛（全部）

**优点：**
- ✅ 最多投注机会
- ✅ 包括国际赛事

**新增：**
- 欧冠、欧联、欧协联
- 世界杯、欧洲杯、美洲杯
- 南美解放者杯、南美杯

## 💡 使用建议

### 阶段一：测试期（推荐 TIER_1）

```python
LEAGUE_MODE = 'TIER_1'
```

**理由：**
1. 质量最高，胜率最稳定
2. 容易收集数据和分析
3. 38个联赛足够找到投注机会
4. 适合验证策略有效性

### 阶段二：扩展期（TIER_1 胜率 > 60%）

```python
LEAGUE_MODE = 'EXPANDED'
```

**条件：**
- TIER_1 已运行 100+ 场
- 真实胜率 ≥ 60%
- 数据分析显示策略有效

**目的：**
- 增加投注频率
- 测试策略在更多联赛的表现
- 找出高胜率的新联赛

### 阶段三：全面期（经验丰富）

```python
LEAGUE_MODE = 'ALL'
```

**条件：**
- EXPANDED 模式已验证
- 有充足的资金和经验
- 理解杯赛和国际赛事的特点

## ⚠️ 重要提醒

### 1. 不同联赛胜率差异大

**可能情况：**
- 五大联赛胜率：60-65%
- 欧洲主流：55-60%
- 亚洲联赛：50-55%
- 非洲联赛：< 50%

**建议：**
- 先用 TIER_1 测试
- 分联赛统计胜率
- 只投注高胜率联赛

### 2. 联赛越多 ≠ 收益越高

**风险：**
- 低质量联赛可能降低整体胜率
- 数据不充分导致判断失误
- 虚假的"强队-1"

**原则：**
- 质量 > 数量
- 先验证再扩展
- 不熟悉的联赛谨慎投注

### 3. 分析每个联赛的表现

运行一段时间后，统计：

```
联赛              场次  胜  负  胜率
英超              20    13  7   65%  ✅
西甲              18    11  7   61%  ✅
巴甲              15    8   7   53%  ⚠️
非洲某联赛        5     1   4   20%  ❌
```

**策略：**
- 胜率 > 60%：继续投注
- 50% < 胜率 < 60%：观察
- 胜率 < 50%：停止该联赛

## 💡 常见问题

### Q1: 为什么默认只有38个联赛？

**A:** 质量 > 数量。TIER_1 的 38 个联赛是信息最充分、最靠谱的。先验证策略有效性，再扩展。

### Q2: 什么时候可以切换到 EXPANDED？

**A:** 满足以下条件：
- TIER_1 运行 100+ 场
- 真实胜率 ≥ 60%
- 资金管理良好

### Q3: ALL 模式包括友谊赛吗？

**A:** 包括国际友谊赛，但友谊赛的-1盘口很少，且质量不高，不推荐。

### Q4: 如何知道哪些联赛表现好？

**A:** 运行后手动统计 `bet_log.csv`，按联赛分组计算胜率。

### Q5: 可以自定义联赛列表吗？

**A:** 可以。编辑 `core_leagues.py`，在对应的 TIER 中添加或删除联赛 key。

|------|--------|---------|------|------|
